Firebase · GitHub 연동 설정 방법 보기
1) Firebase 프로젝트 만들기
console.firebase.google.com → 프로젝트 생성 → Firestore Database 사용 설정(프로덕션 모드) → Authentication에서 이메일/비밀번호 로그인 사용 설정.

2) 웹 앱 등록 후 설정값 복사
프로젝트 설정 → 내 앱 → 웹(</>) 추가 → firebaseConfig 값을 이 파일 상단 FIREBASE_CONFIG에 붙여넣기.

3) Firestore 보안 규칙 (로그인한 관리자만 읽기·쓰기 가능): 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /applications/{doc} {
      allow read, write: if request.auth != null;
    }
  }
}
4) GitHub Pages 배포
이 파일을 index.html로 저장소에 올리고 Settings → Pages에서 배포. 배포 주소를 Firebase Authentication → 설정 → 승인된 도메인에 추가.

5) 최초 실행 시 “관리자 계정 등록하기”로 이메일과 비밀번호(6자 이상)를 한 번만 등록하면, 이후 같은 이메일·비밀번호로 로그인합니다.
